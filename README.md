## Notebook summaries

### pythonFacil_basic_00_built_in_function_python.ipynb
This notebook focuses on Python built-in functions. It explains how to define and call functions, use parameters and return values, and organize code into reusable blocks.

### pythonFacil_basic_01_DataStructures.ipynb
This notebook introduces Python data structures such as lists, tuples, dictionaries, and sets. It includes examples and exercises that show how to store, access, and manipulate structured data effectively.

### pythonFacil_basic-02-operators-if-loops-sol.ipynb
This notebook covers Python operators, conditionals, and loops with solved examples. It is designed to help beginners practice core control flow patterns and simple expression evaluation.

### pythonFacil_basic-03-funcions_lambdas.ipynb
This notebook explores advanced function concepts in Python, including lambda functions and functional programming patterns. It demonstrates how to write concise anonymous functions and use higher-order functions for elegant code solutions.

### pythonFacil-projecte1.ipynb
This notebook is a small project that applies Python skills to a practical task. It demonstrates how to combine learned concepts in a project-style exercise, reinforcing problem solving and code structure.
