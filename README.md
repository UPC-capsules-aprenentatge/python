### pythonFacil-basic-00-FuncionsPython..ipynb
This notebook focuses on Python functions. It explains how to define and call functions, use parameters and return values, and organize code into reusable blocks.

### pythonFacil-basic-01-DataStructures.ipynb
This notebook introduces Python data structures such as lists, tuples, dictionaries, and sets. It includes examples and exercises that show how to store, access, and manipulate structured data effectively.

### pythonFacil- basic-02-operators-if-loops-sol.ipynb
This notebook covers Python operators, conditionals, and loops with solved examples. It is designed to help beginners practice core control flow patterns and simple expression evaluation.

### pythonFacil-basic-002-2-operators-if-loops.-sol.ipynb
This notebook provides a second set of operator and loop exercises, with a focus on solving problems using if statements and iteration. It reinforces the same fundamentals through additional examples and solutions.
